# gomarcd.dev

This is the [gomarcd.dev](https://gomarcd.dev) website, made with the static site generator [Hugo](https://gohugo.io) and [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme.

# WoodpeckerCI

It's deployed using [WoodpeckerCI](https://woodpecker-ci.org) so that when a change is pushed to [the repo](https://codeberg.org/gomarcd/gomarcd-website), Hugo automatically builds the site and the built pages get copied over to the server, quickly pushing the changes into production.
